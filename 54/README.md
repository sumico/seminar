RICE (Remote Sensing Images for Cloud Removal)

「RICE」は光学リモートセンシング画像における 雲の影響を除去する（cloud removal） というタスク向けに作られたベンチマークデータセット

RICE is a benchmark dataset designed for the task of cloud removal in optical remote sensing images.

Source: https://www.kaggle.com/datasets/shubhank001/rice-remote-sensing-images-for-cloud-removal
